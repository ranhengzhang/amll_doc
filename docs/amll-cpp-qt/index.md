# AMLL C++ Qt 项目

基于 C++ Qt 框架开发的 AMLL 歌词工具集合，提供 TTML 歌词格式的解析、转换和导出功能。

## 项目列表

- [LyricParser](./lyric-parser.md) - C++ 歌词格式解析与转换库
- [TTML Trans Tool](./ttml-tool.md) - TTML 歌词转换桌面工具
- [TTML Converter](./ttml-converter.md) - TTML 字幕命令行转换器与右键菜单工具

## 技术栈

| 组件 | 技术 |
|------|------|
| **语言** | C++23 |
| **框架** | Qt 6.x |
| **构建系统** | CMake 3.20+ |

## 支持的歌词格式

| 格式 | 说明 |
|------|------|
| TTML | Apple Music / 时序文本标记语言 |
| ASS | Advanced SubStation Alpha（Aegisub / 卡拉 OK） |
| LRC | 标准歌词格式 |
| SPL | Salt Player Lyric |
| LYS | 专用歌词格式 |
| QRC | QQ 音乐歌词格式 |
| YRC | 网易云音乐歌词格式 |
| KRC | 酷狗音乐歌词格式 |
| TXT | 纯文本 |
